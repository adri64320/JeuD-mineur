readme Exercice TD 9, BERNARD Adrien Groupe 1 ING 1 GI

Compilation : make
Les fichiers .o et l'executable sont dans le dossier bin

Utilisation : make exe

Generer la documentation Doxygen : make doc 
La documentation se trouve dans le dossier doc

Génerer une archive : make give

Supprimer les fichiers génrable : make clean

Sauvegarder et restaurer la sauvegarde : make save 
                                         make restore
Les fichiers se trouvent dans le dossier save
